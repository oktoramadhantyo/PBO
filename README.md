# Kalkulator Java

Proyek ini berisi sebuah program kalkulator sederhana yang ditulis dalam Java.

## File

- `Kalkulator.java` - sumber program utama.

## Cara Menjalankan

1. Buka terminal di folder `a`.
2. Kompilasi program:
   ```bash
   javac Kalkulator.java
   ```
3. Jalankan program:
   ```bash
   java Kalkulator
   ```

## Keterangan

Program ini dapat digunakan untuk melakukan operasi dasar pada bilangan, tergantung pada implementasi yang tersedia di `Kalkulator.java`.
